# Founder Predictors (summary)

> [!note] Full version
> The complete writeup with the do-NOT-adopt list lives in the repo at `docs/research/founder-predictors.md` (kept there so judges see it as an Area-of-Research deliverable). This note is the brain-side summary. Wired into [[intelligence-layer]] via the Ambition & Drive read.

**Premise:** ideas change, most startups pivot — so evaluate qualities that survive a pivot, observable from a public footprint.

| Predictor | Strength | Source | In Argus |
|---|---|---|---|
| Determination / relentless resourcefulness | Strongest (YC) | [Paul Graham](https://www.paulgraham.com/relres.html) | Resourcefulness rubric + ambition read |
| Learning velocity (tried→failed→changed) | Strong practitioner consensus | [CRV](https://www.crv.com/content/what-seed-investors-look-for), [VC Cafe](https://www.vccafe.com/betting-on-people-what-vcs-look-for-at-the-pre-seed-stage/) | Momentum rubric + ambition read |
| Calibrated ambition vs hype | Qualitative but decisive | practitioner lit | `hypeRisk` in ambition read |
| Track-record persistence (34% vs 22%) | Solid data | [Gompers et al. JFE 2010](https://www.hbs.edu/faculty/Pages/item.aspx?num=37618), [NBER](https://www.nber.org/papers/w12592) | Positive-only signal (no first-timer penalty) |
| Resilience across stages | Moderate | [PNAS 2023](https://www.pnas.org/doi/10.1073/pnas.2215829120), [Nature 2023](https://www.nature.com/articles/s41598-023-41980-y) | Persistence evidence in ambition read |
| Personality traits | Weak (~10% variance) | [Columbia](https://business.columbia.edu/research-brief/research-brief/startups-founder-personalities-vc) | Deliberately NOT scored |

**Rejected on mission grounds:** solo-founder penalty, first-timer penalty, personality testing from deck text — each would rebuild the network-gated bias ([[decisions]] #9).
